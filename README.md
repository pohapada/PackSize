# برنامه محاسبه ابعاد کارتن

یک برنامه تحت وب پیشرونده (PWA) برای محاسبه ابعاد انواع کارتن بر اساس فرمول‌های سفارشی.

## ویژگی‌ها

- محاسبه ابعاد ۴ نوع کارتن: کیبوردی، کارتن مادر، ته قفلی، کلافی+پیتزایی
- قابلیت نصب روی دستگاه (PWA)
- امکان تغییر فرمول‌های محاسباتی
- ذخیره تنظیمات در مرورگر
- طراحی واکنش‌گرا
- پشتیبانی از زبان فارسی

## نحوه استفاده

1. مقادیر طول، عرض و ارتفاع را وارد کنید
2. روی دکمه "محاسبه ابعاد" کلیک کنید
3. نتایج برای هر نوع کارتن نمایش داده می‌شود

## فرمول‌های پیش‌فرض

### کیبوردی
- عرض شیت: `3 + طول + (4 × ارتفاع)`
- طول شیت: `2 + (2 × عرض) + (3 × ارتفاع)`

### کارتن مادر
- عرض شیت: `طول + ارتفاع`
- طول شیت: `4cm + 2 × (طول + عرض)`

### ته قفلی
- عرض شیت: `5cm + 0.5 عرض + ارتفاع + عرض`
- طول شیت: `2 × (طول + عرض) + 4cm`

### کلافی+پیتزایی
- عرض شیت: `2cm + 2 × ارتفاع + طول`
- طول شیت: `4 × ارتفاع + 2 × عرض`

## نصب برنامه

برای نصب برنامه:
1. در مرورگر باز کنید
2. روی دکمه "نصب برنامه" کلیک کنید
3. دستورالعمل‌های نصب را دنبال کنید
